# 코드 구조 요약

## 프로젝트 구조

```
Boardgame/
  app/
    main.py           # FastAPI 진입점 (/, /health). Docker에서 uvicorn으로 실행.
  games/
    _template/
      game.yaml       # 게임 설정 템플릿. 새 게임 시 games/{game_id}/game.yaml 로 복사.
  docs/
    prompts/          # LLM 프롬프트 템플릿 (파이프라인 1·2단계)
      01-룰-공통요소정리.md   # 룰(자연어) → 공통 요소 정리 (사용자 검수용)
      02-정리내용-코드생성.md # 검수된 정리본 → 플러그인 코드 생성
    TODO.md
    보드게임플랫폼기술_방향성.md
    코드 구조 요약.md  # 이 문서
```

- **Docker**: `docker-compose up --build` 로 app(Python API) + db(PostgreSQL) 실행. 엔진은 앱 밖/추가 예정.
- **파이프라인**: 룰(자연어) → 1단계 프롬프트(공통 요소 정리) → 사용자 검수 → 2단계 프롬프트(플러그인 코드 생성). IR(JSON) 중간 변환 없음. `docs/prompts/` 참고.
