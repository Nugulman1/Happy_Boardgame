# 코드 구조 요약

## 프로젝트 구조

```
Boardgame/
  app/
    main.py           # FastAPI 진입점 (/, /health). Docker에서 uvicorn으로 실행.
  schema/
    README.md         # IR 형식·경로·템플릿 사용법 (기술)
    game-ir.schema.json  # IR JSON Schema (개념 정한 뒤 확장)
  games/
    _template/
      game.yaml       # 새 게임 추가 시 복사용 템플릿
  docs/
    TODO.md
    보드게임플랫폼기술_방향성.md
    코드 구조 요약.md  # 이 문서
```

- **Docker**: `docker-compose up --build` 로 app(Python API) + db(PostgreSQL) 실행. IR 완성본·엔진은 앱 밖/추가 예정.
- **IR**: 완성본은 `games/{game_id}/game.yaml` 에 두는 것을 전제. 초안·검수는 별도 도구/프롬프트.
