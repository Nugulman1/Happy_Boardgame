# 4단계: 트릭 루프 + 라운드 종료

## 목표
- 라운드 내 여러 트릭을 반복 실행한다.
- 원투 즉시 종료 및 일반 종료(3명 소모)를 정확히 판정한다.

## 신규 함수

### `tichu/loop.py` (또는 `tichu/game_loop.py` 내부)
- `advance_turn(round_state: RoundState) -> None`
- `is_round_over(round_state: RoundState) -> bool`
- `run_trick_phase(round_state: RoundState) -> None`

## 구현 규칙
- 턴 진행은 `current_player_index` 기준 시계방향 순환.
- 손패가 0장인 플레이어는 턴 후보에서 스킵.
- 플레이어 손패가 0장이 되는 즉시 `finish_order`에 최초 1회 추가.
- 종료 판정:
  - 원투: `finish_order` 2명이 같은 팀이면 즉시 라운드 종료.
  - 일반: `finish_order` 길이 3이면 라운드 종료.
- 라운드 종료 직전 꼴찌는 `finish_order`에 자동 확정 추가.

## 트릭 루프 흐름
1. 현재 선 플레이어로 트릭 시작.
2. 플레이/패스 반복.
3. 3패스면 트릭 종료 및 다음 선 갱신.
4. 라운드 종료가 아니면 다음 트릭으로 반복.

## 테스트 케이스
- 2~3개 트릭 연속 진행 시 상태 전이 정상 동작.
- 원투 발생 즉시 종료 확인.
- 원투 미발생 상태에서 3명 소모 종료 확인.
- 손패 0장 플레이어 턴 스킵 검증.

## 단계 완료 기준
- 임의 라운드 시뮬레이션에서 종료 조건 2종이 오판정 없이 동작해야 한다.
