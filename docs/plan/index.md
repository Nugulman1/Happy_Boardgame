## 공통 결정사항
- 일반 라운드 점수 정산은 공식형으로 고정한다.
  - 마지막 플레이어 손패 점수는 상대팀(1등 팀)으로 이동.
  - 마지막 플레이어가 확보한 트릭 점수는 1등 플레이어에게 이동.
- 폭탄 3초 유예는 실시간 타이머가 아닌 턴기반 근사 응답 단계로 구현한다.
- 단계별 구현은 "입력 수집 -> 유효성 검사 -> 상태 적용" 순서를 유지한다.

## 단계별 완료 조건
- 3단계: 족보/폭탄 판정 + 트릭 기본 흐름(선 제출, 덮기, 패스, 3패스 종료) 동작.
- 4단계: 트릭 루프 완료 + 원투/3인 소모 라운드 종료 판정 동작.
- 5단계: 라운드 점수 정산 + 게임 1000점 종료 루프 동작.
- 6단계: 스몰 티츄, 참새 소원/콜, 폭탄 유예 근사, 용/개 특수 규칙 동작.
