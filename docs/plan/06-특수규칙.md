# 6단계: 특수 규칙 순차 추가

## 목표
- 핵심 플레이 규칙에 영향을 주는 특수 규칙을 백엔드에서 완결한다.

## 구현 항목

### 1) 스몰 티츄
- 상태: `RoundState.declared_tichu: set[int]` 사용.
- 규칙:
  - 해당 플레이어 첫 카드 제출 전까지만 선언 가능.
  - 성공(1등) +100, 실패 -100.
- 함수:
  - `declare_tichu(round_state: RoundState, player: int) -> bool`
  - `is_tichu_success(player: int, finish_order: list[int]) -> bool`

### 2) 참새 소원/콜
- 상태: `RoundState.wish_rank: int | None`.
- 규칙:
  - 참새를 낸 순간 `wish_rank` 설정.
  - 손패에 있고 현재 조합에 포함 가능하면 소원 숫자 포함 강제.
  - 소원 가능한 조합이 있으면 패스 불가.
- 함수:
  - `set_wish_on_mahjong_play(round_state: RoundState, wished_rank: int) -> None`
  - `must_follow_wish(round_state: RoundState, player: int) -> bool`

### 3) 폭탄 유예 (턴기반 근사)
- 상태: `RoundState.bomb_response_open: bool`.
- 규칙:
  - 일반 제출 직후 폭탄 응답 단계 1회 오픈.
  - 응답 가능한 플레이어가 더 강한 폭탄을 제출하면 폭탄 체인 허용.
  - 더 이상 응답 없으면 기존 트릭 진행 재개.
- 함수:
  - `open_bomb_response(round_state: RoundState) -> None`
  - `resolve_bomb_response(round_state: RoundState) -> None`

### 4) 용 점수 전달
- 규칙:
  - 용으로 트릭 획득 시 해당 트릭 점수 카드를 상대팀 플레이어 1명에게 전달.
- 함수:
  - `transfer_dragon_trick_points(round_state: RoundState, winner: int, target_enemy: int) -> None`

### 5) 개(Dog) 처리
- 규칙:
  - 개를 내면 점수 없는 특수 행동으로 처리.
  - 즉시 팀원에게 선턴 이전, 현재 트릭은 종료 처리.
- 함수:
  - `apply_dog_play(round_state: RoundState, player: int) -> None`

## 테스트 케이스
- 스몰 티츄 선언 가능 시점/성공/실패 점수.
- 참새 소원 강제 및 패스 금지 케이스.
- 폭탄 응답 단계 연쇄 처리.
- 용 점수 전달 대상 검증(상대팀 제한).
- 개 처리 후 선턴 팀원 이전 검증.

## 단계 완료 기준
- 특수 규칙이 기본 트릭/점수 루프와 충돌 없이 작동하고, 회귀 테스트가 1~5단계를 깨지 않아야 한다.
