## 완료

- 봉황 1차 재도입:

  - 폭탄 vs 일반 족보, 폭탄 vs 폭탄 비교 규칙 반영
- 게임 메인 while 뼈대 추가 (`run_game`, `is_game_over`)

  - `resolved_ranks` 기반 봉황 대체 포함 콜 충족 판정
