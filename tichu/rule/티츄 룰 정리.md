# 01-룰-공통요소정리

## 1. 게임 개요

- **게임명:** 티츄 (Tichu)
- **인원:** 4명 고정 (2:2 팀전)  
  ※ 3인/6인/다인 변형 언급은 있으나 본 구현에서는 사용하지 않음
- **목표:** 팀 점수 **1000점**을 먼저 달성

---

## 2. 플레이어

- **역할/팀:** 2대2 팀전, **맞은편끼리 한 팀**
- **턴 순서:** 좌석 기준 고정 순환  
  (자신 → 상대 → 아군 → 상대 → …)
- **선(先) 개념:**
  - 카드 교환 이후 **새 카드를 쥔 사람이 선**
  - 이후 트릭에서는  
    **강패를 내고 모든 패스를 유도해 트릭을 잡은 플레이어가 선**

---

## 3. 게임 상태

- **보드/맵:** 해당 없음 (카드 게임)
- **자원:** 해당 없음
- **카드/토큰:**
  - **덱 구성:** 총 56장  
    (트럼프 52장 + 특수 카드 4장)
  - **특수 카드 4종**
    - 참새(마작/1)
    - 봉황(조커)
    - 용
    - 개
  - **족보(메이드)**
    - 싱글
    - 페어
    - 연속된 페어
    - 트리플
    - 풀 하우스
    - 스트레이트 (최소 5장)
  - **폭탄**
    - 포카드 (동일 숫자 4장)
    - 스트레이트 플러시 (5장 이상 연속 + 동일 무늬)
- **점수/승점**
  - 5 → 5점
  - 10 → 10점
  - K → 10점
  - 봉황 → -25점
  - 용 → +25점
  - 라운드 종료 시 **양 팀 점수 합은 항상 100점**

---

## 4. 턴 / 라운드 구조

- **턴 구성:**  
  패스 포함, 카드를 낼 기회가 돌아오는 단위
- **라운드(Round):**  
  꼴찌를 제외한 **3명이 모든 카드를 소모할 때까지**
- **트릭 / 페이즈**
  1. 선 플레이어가 카드 조합을 내며 시작
  2. 이후 플레이어는
     - 같은 종류의 더 높은 족보를 내거나
     - 패스
  3. **3명이 패스하면**
     - 마지막으로 낸 플레이어가 트릭을 잡고 점수 카드 획득
     - 해당 플레이어가 다음 트릭의 선
  4. 트릭을 잡은 플레이어가 이미 패를 모두 소모했다면
     - 그 다음 플레이어가 선을 이어받음
- **특수 단계**
  - 준비 단계: 카드 배분 / 선언 / 교환
  - 시작 단계: 참새 보유자가 첫 트릭 시작

---

## 5. 행동(액션)

### 기본 행동

#### 1) 카드 내기
- **선 플레이어**
  - 싱글 / 페어 / 연속 페어 / 트리플 / 풀 하우스 / 스트레이트(5장 이상)
- **이후 플레이어**
  - 직전 카드와 **같은 종류의 더 높은 족보**
  - 또는 패스
- 스트레이트 / 연속 페어는 **장수 동일**해야 덮기 가능

#### 2) 패스
- 패스해도 트릭에서 영구 이탈하지 않음
- 다음 차례에 다시 카드 제출 가능

#### 3) 폭탄 사용
- 트릭이 활성 상태라면 **턴과 무관하게 사용 가능**
- 단, 다음 제한이 존재:
  - 트릭의 **첫 카드가 나오기 전(맨바닥)**에는  
    선 플레이어만 카드 제출 가능
  - 개 사용 중에는 폭탄 사용 불가
  - 폭탄은 봉황과 조합 불가
- **폭탄 제출 시**
  - 즉시 확정되지 않음
  - **3초 유예시간** 동안 더 강한 폭탄으로 덮기 가능

#### 4) 교환
- 총 14장 수령 후
- 자신을 제외한 3명에게 카드 1장씩 교환

#### 5) 선언
- **스몰 티츄**
  - 자신의 첫 카드 제출 전까지 언제든 선언 가능
  - 성공 +100 / 실패 -100
- **라지(그랜드) 티츄**
  - 최초 8장 상태에서만 선언 가능
  - 성공 +200 / 실패 -200

#### 6) 소원 / 콜 (참새)
- 참새(1)를 **낸 순간에만** 숫자 콜 가능
- 콜 숫자: 2 ~ A (특수 카드 제외)
- 손에 있고, 조합에 포함시킬 수 있다면 **반드시 포함**
- 콜은 **라운드 종료 및 점수 계산 완료 시점까지 유지**
- 콜을 만족하는 조합을 낼 수 있다면 **패스 불가**

---

## 6. 종료 · 승리 조건

- **게임 종료:** 한 팀이 누적 1000점 도달
- **승리 조건:** 1000점에 먼저 도달한 팀
- **라운드 종료**
  - 같은 팀에서 **1등 + 2등(원투)** 발생 시
    - 라운드 즉시 종료
    - 트릭/카드 점수 전부 무시
    - 기본 200점 + 티츄 점수만 가감
  - 그 외에는 3·4등까지 진행 후 점수 계산

---

## 7. 초기 설정

- **카드 배분**
  1. 각자 8장 배분 (즉시 확인 가능)
  2. 이 시점에서 **라지 티츄 선언 / 비선언 선택**
     - 선언 선택에는 **일반 행동과 동일한 타임아웃** 적용
  3. 모든 선택이 확정되면
     - 추가 6장이 **즉시 손패에 합류**
     - 손패는 항상 비공개
  4. 이후 카드 교환 진행
- **선 플레이어 결정**
  - 참새(마작/1) 보유자

---

## 8. 특이사항

- 같은 종류의 족보만 덮을 수 있음
- 백 스트레이트 없음
- 일반 플러쉬 족보 없음
- 폭탄은 트릭이 종료되어 **회수 단계로 넘어간 이후에는 사용 불가**
- **용으로 트릭을 잡은 경우**
  - 해당 트릭의 점수 카드를
  - 용을 낸 플레이어가 **상대 팀 플레이어 1명에게 선택 전달**
